## ru
Добавляет таймер, рестарт игры, возрождения и другие фичи для лучшего опыта выживания/хардкора в Minecraft.
Создан как личный проект, поэтому требует доработок, например, локализацию, сохранение игры, рефакторинг кода.
### админ-команды
* .restart - принудительный перезапуск игры (с использованием настроек команд .hard и .rescue)
* .time - установить время таймера
* .respawn <player> - принудительно возродить игрока
* .hardcore - переключить режим игры (хардкор/выживание)
* .adv - установить достижение для спидрана
* .item - установить предмет для спидрана
* .hard - переключить режим хардкора
* .rescue - переключить возрождения в игре
